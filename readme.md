Homebrew Tap for send2trash
===========================

This repository is a Homebrew tap for the [send2trash](https://github.com/muonium-ai/send2trash) project.

Install
-------
```sh
brew tap muonium-ai/send2trash
brew install send2trash
```

Upgrade
-------
```sh
brew update
brew upgrade send2trash
```

More information
----------------
See the upstream project documentation: https://github.com/muonium-ai/send2trash

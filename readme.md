Homebrew Tap for trash-swift
============================

This repository is a Homebrew tap for the [trash-swift](https://github.com/muonium-ai/trash-swift) project.

Install
-------
```sh
brew tap muonium-ai/trash
brew install trash
```

Upgrade
-------
```sh
brew update
brew upgrade trash
```

More information
----------------
See the upstream project documentation: https://github.com/muonium-ai/trash-swift
